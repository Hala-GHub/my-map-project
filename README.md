# 🗺️ My Favorite Places Map (Calgary)

##  Project Description

This project is a Python application that creates an interactive map of my favorite places in Calgary using the Folium library.

It demonstrates Object-Oriented Programming (OOP) concepts such as:

* Encapsulation
* Inheritance
* Polymorphism
* Abstraction

---

##  Places Included

* Calgary Tower (Landmark)
* Glenbow Museum (Museum)
* Native Tongues Taqueria (Restaurant)
* Prince's Island Park (Park)

---

##  Features

* Interactive map with markers
* Different colors for each place type:

  * 🔵 Place (default)
  * 🔴 Restaurant
  * 🟢 Park
  * 🟣 Museum
* Clickable popups with information
* Distance calculation between places
* Automatically opens in browser

---

##  Technologies Used

* Python
* Folium
* Object-Oriented Programming (OOP)

---

##  How to Run

1. Install required package:

   ```
   pip install folium
   ```
2. Run the program:

   ```
   python my_map.py
   ```
3. The map will open in your browser.

---

##  Output

The program generates an HTML file:

```
my_favorite_places.html
```
